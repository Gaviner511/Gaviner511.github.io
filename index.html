Problem

Job applications are repetitive, inconsistent, and time-consuming. Candidates often re-enter the same information across platforms that use different fields and labels.

What I Built

I designed and developed a Chrome-based workflow automation platform that parses resumes into reusable candidate profiles and supports guided autofill across job application pages.

Key Features

  • Resume parsing into structured candidate profiles
  • Reusable profile export/import
  • Application page scanning
  • Rule-based and AI-assisted field matching
  • Guided autofill with user confirmation
  • Support for complex and ambiguous fields
